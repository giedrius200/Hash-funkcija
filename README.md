Naudojamos bitset XOR operacijos, pakeičiamos bitų vietos
```
bitset<8> bs4(s[i]);
bitset<8> bar(stringas[i]);
bitset<8> bs3(bs4 <<= 2);
bitset<8> rez(bs3 ^= bar);
```
